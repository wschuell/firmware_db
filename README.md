## How to use firmware_db

maintain a small DB of the firmwares of a group of workstations/servers

The project setup is documented in [project_setup.md](project_setup.md). Feel free to remove this document (and/or the link to this document) if you don't need it.

## Installation

To install firmware_db from GitHub repository, do:

```console
git clone git@github.com:wschuell/firmware_db.git
cd firmware_db
python -m pip install .
```
